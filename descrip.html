Qué es Pixel Game Maker MV

Pixel Game Maker MV es una herramienta de desarrollo de videojuegos diseñada para facilitar la creación de videojuegos 2D. Con una interfaz intuitiva y accesible, permite a usuarios de diferentes niveles de experiencie, desde principantes hasta desarrolladores avanzados, crear una amplia variedad de juegos sin necesidad de conocimientos profundos de programación. Este software es ideal para el desarrollo de videojuegos de plataformas, acción y aventura.

Tipos de juegos que puedes crear

Este programa puede crear dos estios de juegos de acción en 2D: top-down (vista desde arriba) y side-scrolling (desplazamiento lateral). Puedes crear juegos de distintos géneros, como juegos de disparos combinando varios elementos de juego. También puedes implementar modo multijugador en offline (VS y o-op) para hasta 4 jugadores

Requisitos del sistema

Sistema operativo Windows® 8/8.1/10 (64bit)
CPU Intel Core2 Duo 2.0GHz o mejor
Memoria 4 GB RAM
Almacenamiento 1 GB+ de espacio libre
Tarjeta gráfica GPU compatible con DirectX 9/OpenGL 4.1
