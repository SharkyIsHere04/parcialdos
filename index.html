Pixel Game Maker MV es un software de desarrollo de videojuegos 2D creado por KADOKAWA.

El software se abrevia como PGMMV en inglés. PGMMV fue lanzado por Gotcha Gtcha Games, una subsidiaria de KADOKAWA como una version beta en 2018. Se lanzó por completo en 2019.

Desarrollo

El desarrollo de Pixel Game Maker gue firigido por Takuya Hatakeyama para Kadokawa Corporation en Tokio. El equipo de desarrollo incluyó personas que habían trabajado en juegos de consola para Game Boy Advance y Nintendo 3DS . El software estaba destinado a facilitar el desarrollo de juegos independientes sin la necesidad de programación. Pixel Game Maker MV fue desarrollado teniendo en mente a los desarrolladores independientes, y sus características se introdujeron gradualmente haciendo referencia a los juegos de acción 2D tradicionales. Durante el desarrollo, el equipo intentó recrear juegos de acción clásicos como Super Mario Bros. y Mega Man para asegurarse de que se incluyeran la mayoría de las características importantes necesarias para desarrollar dichos juegos.

Takuya Hatakeyama informa que durante el desarrollo hubo que solucionar algunos problemas, como el uso excesivo de recursos del software o el recorte de objetos. Estos problemas se solucionaron en la medida de lo posible, pero afirmó que aún pueden ocurrir en cierta medida. Para evitar estos problemas, se incorporaron a Pixel Game Maker MV varias funciones de ajuste para varios parámetros, de modo que los usuarios puedan solucionar estos problemas durante el proceso de desarrollo de un juego.

El software se envió con la inclusión de una serie de funcionalidades de calidad de vida, como tutoriales y la función "ejecutar acción de objeto" que no solo permite a los usuarios iniciar la acción de un objeto al pasar por alto otros enlaces y condicionales, sino que también les permite aplicar la acción de un objeto a otro para que puedan usar la misma acción para múltiples objetos. Otros parámetros, como la gravedad y los ajustes de animación cuadro por cuadro, se pueden incorporar a diseños específicos. La versión de acceso temprano de PGMMV fue lanzada recientemente por Kadokawa Corporation como una versión beta el 24 de julio de 2018. Se lanzó en Steam el 19 de septiembre de 2019.
